# Java_probafeladat

## Funkciók
- felhasználói fiókok (Név) létrehozása, módosítása, törlése
- főmenü kialakítása
- főmenüből nyíló almenü kezelése
- új ikon (Név) hozzáadása, módosítása, törlése
- alkalmazás elindítása
- háttérkép hozzáadása (Név)
- háttérkép kiválasztása (Név)
- arculatváltás (Név)

## Technológiák
- Java 11
- Hibernate 6.3
- H2 adatbázis
- Maven

## Futtatás
1. Indítsd el a H2 szervert
3. Futtasd a Main osztályt IntelliJ-ből

## Struktúra
- `model/` - JPA entitások
- `service/` - üzleti logika
- `Main.java` - belépési pont, felhasználó és kedvenc alkalmazások hozzáadása
